XandeumInsight Pro
XandeumInsight Pro is a high-fidelity analytics engine built for institutional-grade monitoring of the Xandeum pNode network.

Key Analytics Modules
Institutional Dashboard: High-density UI optimized for monitoring network health and cluster load.

Propagation Indexing: Real-time tracking of gossip endpoints and validator status.

Topology Visualization: Visual mapping of node connectivity across the Xandeum ecosystem.

Deployment
. Install dependencies: npm install.

. Build for production: npm run build.

. Deployed via Vercel for 99.9% uptime.
